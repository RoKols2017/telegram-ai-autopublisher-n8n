# Portfolio Case Study

## Кейс

Этот проект оформляет учебные задания по Prompt Engineering 2.0 как production-style AI automation case. Вместо абстрактной демонстрации отдельных промптов репозиторий показывает полный workflow: защищённый webhook, multi-step LLM pipeline, генерацию изображения и реальную публикацию в Telegram.

## Что показывает проект

- умение проектировать application-layer workflow поверх готовой self-hosted infrastructure
- умение интегрировать OpenAI API и Telegram Bot API через `n8n`
- умение проектировать prompts под разные задачи в одном pipeline
- умение учитывать безопасность webhook и hygiene экспортируемых workflow
- умение упаковывать инженерную работу в понятный GitHub portfolio case

## Engineering angle

- clear scope split between infrastructure and application layers
- sanitized artifacts for public GitHub publication
- repeatable testing through Python scripts
- bilingual documentation for portfolio visibility

## Result

Репозиторий позиционирует автора как AI Automation & LLM Solutions Engineer, который не только экспериментирует с prompt engineering, но и доводит его до работающей automation-system integration.
