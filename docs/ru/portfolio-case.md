# Portfolio Case

## Что показывает этот MVP

Этот репозиторий позиционирует автора как инженера, который умеет:

- строить application-layer AI automation workflows
- интегрировать OpenAI API и Telegram Bot API через `n8n`
- проектировать prompts для multi-step generation
- разделять infrastructure layer и workflow layer
- оформлять инженерную работу как GitHub portfolio case

## Почему это сильный portfolio MVP

- workflow оформлен как production-style integration
- безопасность и sanitization описаны как обязательная часть процесса
- в репозитории есть тестовые скрипты, troubleshooting и deployment notes
- структура репозитория ориентирована на hiring signal и reusable engineering artifacts
